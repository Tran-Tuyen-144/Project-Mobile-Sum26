## Mục đích
Ứng dụng hỗ trợ cả hai luồng đặt pet:

- **Đặt online**: khách hàng tự đặt pet qua app trước khi đến.

Tài liệu chi tiết về chức năng Đặt Pet Offline được tách riêng trong [OFFLINE_BOOKING.md](OFFLINE_BOOKING.md).
